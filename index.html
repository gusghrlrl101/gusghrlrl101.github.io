💻 프로젝트

무인 상점 프로젝트

  • 실시간 멀티카메라 사람 트래킹 및 위치 추정을 이용한 무인 상점 개발 @ if Kakao 발표
    • Deep Stream, Tensor RT, Jetson, Gstreamer, C++, CUDA, Python, Influxdb, Kafka
  • 무인 상점 프로젝트는 여러 대의 카메라를 통해 사람의 동선과 위치를 실시간으로 파악하는 시스템입니다. Amazon Go와 같은 무인 상점 서비스는 3D depth 카메라를 사용하는 경우가 많은데, 일반 RGB 카메라를 이용해 구현하는 게 목표였습니다. Nvidia Jetson (엣지 디바이스) 환경에서, 여러 카메라로부터의 스트림 처리 및 비디오 인코딩을 위해 C++ 기반의 Deep Stream을 사용했습니다. 실시간 처리를 위해 딥러닝 모델 추론 속도가 빠른 Tensor RT로 변환 및 추론을 진행했습니다. 최적화가 필요한 부분에서는 CUDA를 활용해 성능을 향상했습니다. 여러 기능의 Gstreamer 플러그인을 직접 구현하여 모듈 형태로 구성했습니다. 시계열 데이터 처리를 위해 InfluxDB, Kafka를 활용했습니다. 또한 엣지 디바이스 환경뿐만 아니라, Kubernetes를 사용하여 클라우드 환경으로 확장하였습니다. 전체적인 시스템 구조 설계 및 개발을 혼자 진행하여 힘든 순간도 있었지만, 장/단기적인 목표를 정하고 하나씩 처리하며 성공적으로 마칠 수 있었습니다.

이미지 피쳐 검색 엔진 PaaS

  • HNSW (Hierarchical Navigable Small World) 및 OPQ (Optimized Product Quantization) 기반의 검색 엔진을 Argo Workflow를 활용하여 PaaS 개발
    • Kubernetes, Argo Workflow, Python, Flutter
  • 이미지 피처 검색 엔진은 대용량 이미지 데이터 색인이 가능한 Approximate KNN 엔진입니다. 피쳐 크기가 큰 데이터는 PQ, OPQ를 활용하여 압축하고, 여러 노드에 샤딩하며 HNSW를 이용해 색인을 진행합니다. 이미지 피쳐 검색 엔진은 쇼핑하우 에서 유사 상품 추천을 해주는 등, 여러 서비스에 이용되고 있습니다. 데이터 전처리, 빌드 및 배포 등의 과정은 순차적으로 진행돼야 하는 특성이 있는데, Argo Workflow를 이용해 파이프라인을 구성하여 쉽게 관리할 수 있게 했습니다. 이를 통해, 기존 방식보다 자원을 효율적으로 사용할 수 있게 구성했습니다. 또한 쉽게 사용할 수 있는 Web 환경을 구성하여, 서비스 생성 및 데이터 추가, 진행 상황에 대해 쉽게 파악할 수 있게 구성했습니다.

학습 데이터 레이블링 툴

  • 학습 데이터 레이블링을 위한 Web 기반 툴 개발
    • Python, Flutter, Rust, Vue.js
  • 딥러닝 연구를 진행할 때, 학습에 필요한 데이터의 레이블링을 해야 하는 경우가 많이 있습니다. 이를 위해 Web 기반으로 이미지 관련 데이터 레이블링을 할 수 있는 툴을 개발했습니다. 프론트엔드는 Flutter, Vue.js를, 백엔드는 Python, Rust를 이용했습니다. 사용자로부터 보완점, 추가 기능 등의 요구사항을 적극적으로 반영하여 작업 능률을 향상했습니다. 또한, 이때 필요한 데이터 수집과 전처리 작업 등을 진행하면서 데이터 흐름에 대해 이해할 수 있었습니다.

AI 비전 Open API

  • 여러 AI 모델들을 Open API로 제공 @ 데모 페이지
    • Kubernetes, Python, Argo CD, Github Action, Helm
  • 팀 내에서 연구한 여러 AI 기술을 외부에 Open API로 제공하였습니다. 이 과정에서 안정적인 서비스 제공을 위해 구조를 통일화하고, Kubernetes 환경으로 마이그레이션 하였습니다. 개발 과정에서 필요한 여러 인프라를 Helm Chart를 이용하여 빠르게 구축했습니다. 운영/개발 환경을 완전히 구분하여 개발 및 배포 작업이 가능한 시스템을 구성했습니다. 또한 Github Action을 이용하여 테스트, Docker 이미지를 빌드 및 배포하였고, Argo CD를 이용해 자동 배포하여 CI/CD 환경을 구축할 수 있었습니다. 이를 통해, 사용자에게 직접 제공하는 서비스를 진행할 수 있어서 좋은 경험이 되었습니다.
